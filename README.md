# CasaSync 🏠

CasaSync is a collaborative home management platform designed to help families, couples, and shared households organize their spaces, track household needs, and manage recurring expenses in one place.

## ✨ Features

* User authentication
* Multiple household management
* Room organization
* Shared shopping and household needs lists
* Priority and status tracking
* Monthly expense management
* Activity history
* Responsive web experience

## 🚀 Project Vision

Managing a home often involves scattered notes, forgotten purchases, and disconnected expense tracking.

CasaSync aims to provide a centralized platform where household members can collaborate, track what is needed, monitor expenses, and keep everyone informed.
