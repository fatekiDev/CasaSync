# Problema

## En una casa:

Se olvidan las cosas que faltan.
Nadie sabe quién las anotó.
No existe un lugar centralizado.
Los gastos quedan dispersos.
No se puede hacer seguimiento mensual.

## Solución

Una plataforma donde una familia pueda:

Registrar habitaciones.
Registrar necesidades.
Registrar gastos.
Compartir información con otros miembros.
Llevar historial.
